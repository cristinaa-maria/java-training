Primul Design Pattern pe care l-am ales este unul creațional, mai exact Builder. L-am folosit pentru a face un To do list, unde trebuie introduse obligatoriu acțiunea și statusul în care se află ea, iar opțional sunt importanța și termenul limită până la care trebuie finalizată acțiunea. Programul se adresează tuturor celor care vor să țină evidența a ceea ce au de făcut.
Pentru al doilea Design Pattern am ales Template Method, care funcționează ca un calculator de arii și perimetre pentru diverse figuri geometrice. Acest program se adresează în special copiilor din ciclul gimnazial care vor să-și testeze abilitățile de calcul la geometrie. Poate, de asemenea, să fie folosit pentru calculul necesarului de material pentru diverse activități gospodărești(ex: lungimea de gard necesară pentru a înconjura o porțiune de pământ)
